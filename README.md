## Letterboxd 
este es un projecto para emular letterboxd

## Getting Started

```bash
# run dev server
npm run dev
# run storybook
npm run storybook
# deploy storybook to chromatic 
npm run chromatic 
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
